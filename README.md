## Project Overview
This project presents a comprehensive Exploratory Data Analysis (EDA) of the Titanic dataset. The objective is to analyze passenger information, identify survival
patterns, and generate meaningful insights through data visualization and statistical analysis.

## Project Objectives
- Understand the structure of the Titanic dataset.
- Perform data cleaning and preprocessing.
- Analyze factors affecting passenger survival.
- Create informative visualizations.
- Generate actionable insights from the data.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

## Data Cleaning Performed
- Handled missing values in Age and Embarked columns.
- Removed unnecessary columns where required.
- Verified data quality and structure.
- Prepared the dataset for analysis and visualization.

## Key Findings
- Female passengers had significantly higher survival rates than male passengers.
- First-class passengers were more likely to survive than passengers in lower classes.
- Most passengers were between 20 and 40 years of age.
- Passenger class and fare had a noticeable impact on survival outcomes.
- Gender and passenger class were among the strongest predictors of survival.
